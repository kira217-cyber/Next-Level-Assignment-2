# DevPulse API

Backend issue tracking API built with Node.js, TypeScript, Express and PostgreSQL.

## Features

- User authentication
- JWT authorization
- Role based access
- Issue management
- PostgreSQL raw SQL queries

## API Endpoints

### Auth

- POST /api/auth/signup
- POST /api/auth/login

### Issues

- POST /api/issues
- GET /api/issues
- GET /api/issues/:id
- PATCH /api/issues/:id
- DELETE /api/issues/:id

## Tech Stack

- Node.js
- TypeScript
- Express.js
- PostgreSQL
- NeonDB
